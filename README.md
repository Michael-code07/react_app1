# React App (Vite)

Minimal Vite + React starter created automatically.

How to run (Windows PowerShell):

1. Install dependencies:

```powershell
cd "c:\Users\GOIT\OneDrive\Documents\BU-COS 209 Assignment 5\react-app"
npm install
```

2. Start dev server:

```powershell
npm run dev
```

3. Build for production:

```powershell
npm run build
npm run preview
```

Notes:
- This project uses Vite and the official React plugin.
- If Node/npm is not installed on your machine, install Node.js from https://nodejs.org/ first.
